Sets the employment config of specific employment id.

cancelEmployment

Cancels the employment.
Emits a {CancelEmployment} event.

It will try to collect the remaining assets first, and then delete the employment config.

Only the employment creator can cancel the employment.

function cancelEmployment(uint256 employmentConfigId) external;

Parameters

NameTypeDescription
employmentConfigIduint256The employment config ID to cancel.
