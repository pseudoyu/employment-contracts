EmployWithConfig

Git Source

Inherits: IEmployWithConfig, ReentrancyGuard

Logic to handle the employemnt that the employer can set the employment config for a cooperation.,

Employer can set config for a specific employment, and developer can claim the salary by config id. For SetEmploymentConfig Employer can set the employment config for a specific employment
For claimSalary Anyone can claim the salary by employment id, and it will transfer all available tokens from the contract account to the developer account.
